== Description ==
Widget for quick search of railway tickets online (tickets.ua)

== Installation ==
1. Unzip the file  you downloaded
2. Copy the tickets_railway.php file in your Sidebar Widget folder
3. Enable the plugin in your Plugin administration page.
4. Drag the Railway Tickets on your Sidebar
